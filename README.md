File WAV có phần “header” chứa thông tin như:
- Số kênh (1 = mono, 2 = stereo)
- Tần số lấy mẫu (sample rate: 44.1kHz, 48kHz...)
- Độ phân giải bit (8-bit, 16-bit...)
- vvvv...

1 file WAV:
- Mono (1 channel): chỉ có một kênh âm thanh (thường dùng cho giọng nói, tiết kiệm dung lượng).

- Stereo (2 channel): có hai kênh trái và phải (left/right) → nghe có không gian hơn.
